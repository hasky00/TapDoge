# TapDoge
silly game , super simple ,web browser game where your only goal is to  survive the falling blocks
No downloads , No accounts , just tap and doge
build using single HTML file and deploy in under 3 minutes 

# How to Play
avoid the falling blocks for as long as possible
the longer you survivem the higher yous core 

# Controls (mobile)
tap left isde of the screen --> Move left
tap rights side of the screen --> Move right
or drag your finger to control the player smoothly
# Controls ( Desktop )
clik left/right side of the screen
or click and drag the player

# Game rules
You control a blue circle at the bottom of the screen
Blocks fall from top
if a blocks touches you -> game over
score increases over time
the game gradually gets harder 
your best score is saved locally in your browser 

# How it works
BUild with pure HTYML, CSS and Javascript
uses <canvas> for rendering
No framework
No backend
No dependencies 
Fully mobile responsive
